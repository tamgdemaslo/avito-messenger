# Инструкция по настройке приложения Avito

## Шаг 1: Настройка приложения в Avito

1. Перейдите на [Портал разработчика Avito](https://developers.avito.ru)
2. Войдите в свой аккаунт
3. Перейдите в раздел "Мои приложения" или "API"
4. Найдите ваше приложение или создайте новое

## Шаг 2: Настройка Redirect URI

1. В настройках приложения найдите раздел "Redirect URI" или "URI перенаправления"
2. Добавьте следующий URI:
   ```
   http://localhost:5000/callback
   ```
3. Если вы планируете использовать другой порт или домен, измените URI соответственно
4. Сохраните изменения

## Шаг 3: Проверка Client ID и Client Secret

1. Убедитесь, что в файле `app.py` указаны правильные значения:
   ```python
   AVITO_CLIENT_ID = "HHiafcJb3rn8agxhzM8g"
   AVITO_CLIENT_SECRET = "DusHGZF4gADWpNwIHbeefVNwxqoUXe5i_LQ2_g2o"
   ```

2. Если значения отличаются, обновите их в `app.py`

## Шаг 4: Настройка разрешений (Scopes)

Убедитесь, что ваше приложение имеет следующие разрешения:
- `messenger` - для работы с сообщениями
- `read` - для чтения данных
- `write` - для отправки сообщений

## Шаг 5: Запуск приложения

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Запустите приложение:
   ```bash
   python app.py
   ```

3. Откройте браузер и перейдите на `http://localhost:5000`

## Возможные проблемы

### Ошибка "redirect_uri_mismatch"
- Проверьте, что Redirect URI в настройках Avito точно совпадает с `AVITO_REDIRECT_URI` в `app.py`
- Убедитесь, что нет лишних пробелов или символов

### Ошибка "invalid_client"
- Проверьте правильность Client ID и Client Secret
- Убедитесь, что приложение активно в Avito

### Ошибка "access_denied"
- Проверьте, что приложение имеет необходимые разрешения (scopes)
- Убедитесь, что вы авторизуетесь с правильным аккаунтом
