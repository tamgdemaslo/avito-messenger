# Звуковые файлы

## Необходимые файлы

Поместите следующие звуковые файлы в эту папку:

### 1. `popup-sound-modal.mp3`
**Назначение:** Звук получения нового сообщения  
**Когда воспроизводится:** При получении входящего сообщения (каждые 5 секунд проверка)

### 2. `beautiful-sms-notification-sound.mp3`
**Назначение:** Звук отправки сообщения  
**Когда воспроизводится:** Когда вы отправляете сообщение

---

## Как добавить звуки

### Вариант 1: Загрузить свои файлы
1. Скачайте или создайте нужные mp3 файлы
2. Переименуйте их в:
   - `popup-sound-modal.mp3`
   - `beautiful-sms-notification-sound.mp3`
3. Поместите в папку `static/sounds/`
4. Перезагрузите страницу

### Вариант 2: Использовать синтетические звуки (запасной)
Если файлы не найдены, приложение автоматически использует синтетические звуки, созданные через Web Audio API.

---

## Рекомендации

### Для звука получения (`popup-sound-modal.mp3`):
- Длительность: 0.3-0.5 секунд
- Громкость: средняя
- Тип: мягкий "поп" или уведомление

### Для звука отправки (`beautiful-sms-notification-sound.mp3`):
- Длительность: 0.2-0.4 секунды
- Громкость: средняя/тихая
- Тип: короткий "свист" или подтверждение

## Технические детали

### Формат файлов
- Формат: MP3
- Битрейт: 128-192 kbps
- Частота: 44100 Hz
- Каналы: Stereo или Mono

### Громкость в коде
По умолчанию установлена громкость 50% (`volume = 0.5`)

Для изменения громкости отредактируйте в `static/js/app.js`:
```javascript
receiveSound.volume = 0.5; // Громкость получения (0.0 - 1.0)
sendSound.volume = 0.5;    // Громкость отправки (0.0 - 1.0)
```
