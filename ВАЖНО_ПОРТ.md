# ⚠️ ВАЖНО: Изменён порт!

## Проблема решена!

Порт **5000** был занят (скорее всего AirPlay Receiver на macOS).

## Решение:

Приложение теперь работает на порту **5001**.

### Как запустить:

```bash
cd /Users/ilaeliseenko/Desktop/avito-messenger
python3 app.py
```

### Откройте в браузере:

```
http://localhost:5001
```

или

```
http://127.0.0.1:5001
```

## ⚠️ Не забудьте обновить Redirect URI в Avito!

1. Перейдите на [Портал разработчика Avito](https://developers.avito.ru)
2. Найдите ваше приложение
3. Измените Redirect URI на: `http://localhost:5001/callback`

## Если хотите использовать порт 5000:

Остановите AirPlay Receiver:
1. Откройте **Системные настройки** → **Общий доступ**
2. Отключите **AirPlay Receiver**
3. Измените порт обратно на 5000 в `app.py`
