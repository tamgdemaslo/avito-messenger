# Решение проблем

## Белый экран

Если вы видите белый экран, проверьте следующее:

### 1. Убедитесь, что приложение запущено

```bash
cd avito-messenger
python3 app.py
```

Вы должны увидеть сообщение:
```
Avito Messenger запускается...
Откройте в браузере: http://localhost:5000
```

### 2. Проверьте, что порт 5000 свободен

Если порт занят, измените порт в `app.py`:
```python
app.run(debug=True, host='127.0.0.1', port=5001)  # Используйте другой порт
```

### 3. Откройте правильный URL

Убедитесь, что открываете именно:
- `http://localhost:5000` или
- `http://127.0.0.1:5000`

### 4. Проверьте консоль браузера

Нажмите F12 в браузере и проверьте вкладку "Console" на наличие ошибок JavaScript.

### 5. Проверьте вкладку Network

В DevTools (F12) → Network проверьте, загружаются ли CSS и JS файлы.

### 6. Тестовая страница

Откройте `http://localhost:5000/test` - если видите "Flask работает!", значит сервер работает правильно.

### 7. Проверьте установку зависимостей

```bash
pip3 install -r requirements.txt
```

### 8. Проверьте структуру файлов

Убедитесь, что все файлы на месте:
```
avito-messenger/
├── app.py
├── templates/
│   ├── index.html
│   └── messages.html
└── static/
    ├── css/
    │   └── style.css
    └── js/
        └── app.js
```

## Другие проблемы

### Ошибка "ModuleNotFoundError"

Установите зависимости:
```bash
pip3 install Flask flask-cors requests
```

### Ошибка "Address already in use"

Порт 5000 занят. Измените порт в `app.py` или остановите другой процесс на этом порту.

### CSS не загружается

Проверьте, что файл `static/css/style.css` существует и Flask правильно настроен для статических файлов.
