# Инструкция по запуску (решение ошибки 403)

## Проблема: HTTP ERROR 403

Ошибка 403 означает, что сервер не запущен или вы открываете неправильный адрес.

## Решение:

### Шаг 1: Откройте терминал

Нажмите `Cmd + Space` и введите "Terminal", затем нажмите Enter.

### Шаг 2: Перейдите в папку проекта

```bash
cd /Users/ilaeliseenko/Desktop/avito-messenger
```

### Шаг 3: Запустите приложение

```bash
python3 app.py
```

Вы должны увидеть:
```
==================================================
Avito Messenger запускается...
Откройте в браузере: http://localhost:5000
==================================================
 * Running on http://127.0.0.1:5000
```

### Шаг 4: Откройте в браузере

**ВАЖНО:** Откройте именно этот адрес:
```
http://localhost:5000
```

Или:
```
http://127.0.0.1:5000
```

**НЕ используйте:**
- ❌ `file:///Users/...` (открытие файла напрямую)
- ❌ `http://localhost` (без порта)
- ❌ `localhost:5000` (без http://)

### Шаг 5: Проверьте тестовую страницу

Откройте: `http://localhost:5000/test`

Если видите "Flask работает!" - всё настроено правильно!

## Если всё ещё ошибка 403:

1. **Убедитесь, что Flask запущен** - в терминале должно быть сообщение "Running on..."
2. **Проверьте порт** - возможно, порт 5000 занят, попробуйте изменить на 5001 в `app.py`
3. **Проверьте адрес** - обязательно должен быть `http://localhost:5000` с портом!

## Быстрый запуск (альтернатива):

Используйте скрипт запуска:
```bash
python3 start.py
```
